# Threshold Light — production-treatment checkpoint

All refined note pitches, rhythms, IDs, voices, and staff assignments remain unchanged. This pass
separates attack from sustain with explicit velocity and semantic expression controls, changes
piano pedal at harmonic boundaries, adds a restrained final ritardando, and authors plugin
parameter and mix automation.

The local CLAP manifest selects LedgerLine's deterministic sine reference host. It permits
reproducible timing, cache, routing, and automation tests without downloading an instrument. It is
deliberately not a piano/cello realism claim and must not be silently substituted for approved
production instruments.

Listen for:

- whether CC11 phrase growth continues naturally after each note attack;
- whether pedal changes preserve harmonic clarity and leave the final cadence partially dry;
- whether the cello remains foreground without masking the piano's inner motion;
- whether the ritardando is proportionate;
- after rebinding real instruments, whether gain, EQ, reverb, and articulation mappings still fit.
